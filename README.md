Customer and Product Segmentation using E-Commerce Data

Capstone Project ini menampilkan analisis segmentasi pelanggan menggunakan data RFM (Recency, Frequency, Monetary), (Quantity, Country), (Quantity, Total Price, Unit Price, Invoiceno). Projec ini juga mennggunakan 2 Algoritma yaitu K-Means dan DBscan. Aplikasi dibangun dengan Python, Github dan Streamlit.
